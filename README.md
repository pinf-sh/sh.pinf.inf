sh.pinf.inf
===========

Usage
-----

    nvm use 10
    npm install -g pinf

    echo '#!/usr/bin/env inf
    {
        "pinf @ sh.pinf.0 # genesis.inception": {
            "commands": {
                "open": "echo 'Hello World!'"
            }
        }
    }' > _#_sh.pinf_#_.inf.json

    pinf
